Anime Discovery is a site made for Anime Lovers who want to/are searching for more anime to watch. The Site provide a bunch of Popular Anime in genres: Action, Family, Child & Adventure. The Site is made from scratch and all the animes and descriptions were made manually. I could have used API to fetch and display all that information form another site (TMDB, IMDB NETFLIX), but I felt that would be to easy and for this project I wanted to test my skills by building and designing the site as if the data was fetched using API. Ajax and Javascript are used to make the site more responsive. As for now the User will only be able to see list of animes in genres and a brief description of them. The animes displaying in the header will have a learn more button that will take you to more information about the anime. In the future I will add more fonctionalities so the User will be able To even Download an anime from the site. As For now Please use this Site to see which animes are trending now! And go watch them on a Streaming Site. Thanks!
